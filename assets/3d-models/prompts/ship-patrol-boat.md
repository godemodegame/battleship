# Ship Patrol Boat Prompt

## Runtime Asset

`ship-patrol-boat.glb`

## Prompt

Create an original stylized 3D patrol boat for a mobile Battleship-style game. The ship should fit within one board cell and remain readable at a small size. Use a dark hull, bright cyan rim light, tiny amber signal detail, and painterly animated texture. The style is adult animated neo-noir graphic 3D with ink-like contour edges and high contrast.

The patrol boat should have a compact, punchy silhouette: pointed front, small cabin, short rear deck, and a few angular sci-fi naval details. It should feel like a tactical miniature, not a toy.

## Requirements

- One-cell footprint.
- Clear readable silhouette at small size.
- Original fictional design.
- Minimal geometry.
- Works as repeated asset for multiple one-cell ships.
- No text, flags, or logos.

## Tripo Settings

Use these Tripo generation settings:

- AI Model: `v3.1 - Best Quality`
- Texture: `On`
- Texture Quality: `2K`
- PBR: `Off`
- Topology: `Triangle`
- Polycount: `1200`
- Ultra Mesh Quality: `Off`
- Generate in Parts: `Off`
- 8K Texture: `Off`
- Privacy: `Sharing Only`
- Runtime Target: lightweight mobile WebGL/browser asset with fast loading and low GPU cost

## Negative Prompt

No real patrol boat replica, no existing IP, no exact show design, no cute toy look, no flat low-poly placeholder, no national markings, no readable text, no excessive detail that disappears on mobile.

## Notes

The one-cell ship needs the strongest silhouette clarity of all ship models.
