# Ship Submarine Prompt

## Runtime Asset

`ship-submarine.glb`

## Prompt

Create an original stylized 3D submarine for a mobile Battleship-style game. The submarine should span two board cells and have a low, sleek, hidden silhouette. Use a dark blue-black hull, faint cyan seam lights, minimal amber sensor dots, and painterly roughness. The style is adult animated neo-noir graphic 3D with ink-like contours and dramatic rim lighting.

The submarine should feel secretive and encrypted, like a stealth object under a tactical ocean grid. Give it a smooth elongated body, compact sail, and subtle fin shapes. Keep the model readable from an angled top-down mobile camera.

## Requirements

- Two-cell length silhouette.
- Low-profile underwater feeling.
- Original fictional design.
- Distinct from destroyer.
- Mobile-friendly geometry.
- No text, flags, or logos.

## Tripo Settings

Use these Tripo generation settings:

- AI Model: `v3.1 - Best Quality`
- Texture: `On`
- Texture Quality: `2K`
- PBR: `Off`
- Topology: `Triangle`
- Polycount: `2200`
- Ultra Mesh Quality: `Off`
- Generate in Parts: `Off`
- 8K Texture: `Off`
- Privacy: `Sharing Only`
- Runtime Target: lightweight mobile WebGL/browser asset with fast loading and low GPU cost

## Negative Prompt

No real submarine replica, no existing franchise design, no exact show reference, no national symbols, no readable markings, no cute toy shape, no flat gray realism, no excessive tiny surface details.

## Notes

The submarine can sit slightly lower relative to the grid than surface ships to suggest concealment.
