# Ship Carrier Prompt

## Runtime Asset

`ship-carrier.glb`

## Prompt

Create an original stylized 3D carrier ship for a mobile-first Battleship-style game. The ship should span four board cells and read clearly from an angled top-down camera. Use a dark graphite hull, electric cyan tactical edge lights, small amber deck indicators, and restrained magenta warning accents. The visual style is adult animated neo-noir graphic 3D with ink-like contours, painterly metal roughness, and high contrast lighting.

The carrier should have a broad flat deck, exaggerated command tower, sharp silhouette, and subtle futuristic naval details. It should feel like a dramatic animated war-game miniature, not a real-world military replica. Keep the shape clean, bold, and readable on a phone screen.

## Requirements

- Four-cell length silhouette.
- Strong top-down readability.
- Original ship design.
- Dark hull material with cyan and amber accents.
- No national flags or realistic military markings.
- Clean topology suitable for `.glb`.

## Tripo Settings

Use these Tripo generation settings:

- AI Model: `v3.1 - Best Quality`
- Texture: `On`
- Texture Quality: `2K`
- PBR: `Off`
- Topology: `Triangle`
- Polycount: `4200`
- Ultra Mesh Quality: `Off`
- Generate in Parts: `Off`
- 8K Texture: `Off`
- Privacy: `Sharing Only`
- Runtime Target: lightweight mobile WebGL/browser asset with fast loading and low GPU cost

## Negative Prompt

Do not copy any real aircraft carrier, existing game asset, show asset, logo, or protected design. No photorealistic navy simulator, no tiny unreadable deck clutter, no text, no flags, no toy plastic material, no beige palette.

## Notes

This is the flagship asset. It can have more detail than smaller ships, but it must still be mobile-friendly.
