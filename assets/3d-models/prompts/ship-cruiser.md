# Ship Cruiser Prompt

## Runtime Asset

`ship-cruiser.glb`

## Prompt

Create an original stylized 3D cruiser ship for a mobile Battleship-style game. The ship should span three board cells and have a sleek medium-weight silhouette. Use a dark petrol hull, cyan tactical glow lines, small amber signal panels, and painterly rough metal texture. The style is adult animated neo-noir graphic 3D with ink-like edge contours and sharp neon reflections.

The cruiser should feel fast, precise, and elegant. Give it a long angular hull, compact bridge, subtle antenna shapes, and clean layered armor panels. Avoid realistic national navy details and keep it readable from a mobile top-down camera.

## Requirements

- Three-cell length silhouette.
- Sleek medium ship profile.
- Distinct from battleship and destroyer.
- Original design.
- Mobile-friendly mesh complexity.
- Clean material slots for player and enemy variants.

## Tripo Settings

Use these Tripo generation settings:

- AI Model: `v3.1 - Best Quality`
- Texture: `On`
- Texture Quality: `2K`
- PBR: `Off`
- Topology: `Triangle`
- Polycount: `2600`
- Ultra Mesh Quality: `Off`
- Generate in Parts: `Off`
- 8K Texture: `Off`
- Privacy: `Sharing Only`
- Runtime Target: lightweight mobile WebGL/browser asset with fast loading and low GPU cost

## Negative Prompt

No copied existing ship, no protected IP, no exact show-inspired object, no national flags, no logos, no readable text, no toy style, no plain gray realism, no overly thin fragile details.

## Notes

The cruiser should be visually balanced: more elegant than the battleship, more substantial than the destroyer.
