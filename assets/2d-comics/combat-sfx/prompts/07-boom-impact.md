# BOOM Impact Sprite Prompt

## Runtime Asset

`comic-sfx-boom.png`

## Prompt

```text
Create one isolated 2D comic onomatopoeia result sprite containing exactly the
text "BOOM!" for a strong impact variation in a mobile Battleship game. Use an
original adult animated neo-noir graphic illustration style: high-contrast
painted-comic rendering, bold ink-like contours, painterly roughness, dry-brush
edges, selective halftone grain, and dramatic stepped-animation energy.

Make a compact wide burst with irregular angular lobes, never a smooth round
cloud. The two O letters form the heavy center but remain clearly readable;
split them with cold-white impact cuts rather than turning them into fireballs.
Use impact red #FF3B30 and hot magenta #FF2EA6 across the main letters,
signal amber #FFB000 only in small inner pressure marks, cold white #E8F7FF at
the collision center, a thin electric-cyan #21F4FF outer shock edge, and a
thick near-black #07080D painted contour. Add short black smoke wedges and six
uneven shard sparks close to the silhouette.

The result must feel stronger than the lighter hit variants while preserving
board readability. Transparent RGBA background, 1024x1024 source canvas, exact
spelling "BOOM!", no other text. Original design, not based on an existing
comic, show, game, font, logo, artist, or protected visual identity.
```

## Negative Prompt

```text
No generic orange fireball, no realistic explosion, no mushroom cloud, no
word HIT, no extra text, no misspelling, no speech bubble, no ship, no scene,
no gore, no cute style, no smooth gradient logo, no watermark, no cropped
outline.
```
