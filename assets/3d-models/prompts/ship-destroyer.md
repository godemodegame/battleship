# Ship Destroyer Prompt

## Runtime Asset

`ship-destroyer.glb`

## Prompt

Create an original stylized 3D destroyer ship for a mobile Battleship-style game. The ship should span two board cells and have a narrow, fast, angular silhouette. Use dark graphite metal, cyan edge lights, magenta enemy accent slots, and hand-painted roughness. The style is adult animated neo-noir graphic 3D with bold contour shading and high contrast lighting.

The destroyer should look like a quick strike vessel: sharp prow, compact deck structure, small fins or sensor masts, and a clean readable top shape. It should be fictional and stylized, not a real-world naval replica.

## Requirements

- Two-cell length silhouette.
- Fast and narrow profile.
- Clear top-down readability.
- Original design.
- Low enough detail for mobile rendering.
- No text, flags, or logos.

## Tripo Settings

Use these Tripo generation settings:

- AI Model: `v3.1 - Best Quality`
- Texture: `On`
- Texture Quality: `2K`
- PBR: `Off`
- Topology: `Triangle`
- Polycount: `2200`
- Ultra Mesh Quality: `Off`
- Generate in Parts: `Off`
- 8K Texture: `Off`
- Privacy: `Sharing Only`
- Runtime Target: lightweight mobile WebGL/browser asset with fast loading and low GPU cost

## Negative Prompt

No real destroyer replica, no existing game asset, no protected show design, no readable numbers, no national markings, no low-poly placeholder, no plastic toy material, no overly complex rigging.

## Notes

The destroyer should be easy to distinguish from the submarine by having a surface-ship profile and sharper upper structures.
