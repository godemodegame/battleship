# ZIP Speed Sprite Prompt

## Runtime Asset

`comic-sfx-zip.png`

## Prompt

```text
Create one isolated 2D comic onomatopoeia sprite containing exactly the text
"ZIP!" as a one-frame speed accent beside a projectile in a mobile Battleship
game. Use an original adult animated neo-noir graphic illustration style with
high-contrast painted-comic lettering, sharp ink-like contours, painterly
dry-brush breakup, sparse halftone grain, and nervous stepped-animation
energy.

Make the silhouette extremely narrow, diagonal, and forward-leaning from lower
left to upper right. Cut the Z into two razor-like speed planes, compress the I
into a bright needle, and taper the P into the exclamation mark. Use a
cold-white #E8F7FF core, electric-cyan #21F4FF leading edge, tiny hot-magenta
#FF2EA6 chromatic offset, and a thick near-black #07080D contour. Add four very
short parallel ink slashes behind the word, with transparent gaps between
them. The sprite should feel visible for only one or two rendered frames.

Transparent RGBA background, generous clear padding, 1536x768 source canvas,
exact spelling "ZIP!", no other text. It must remain readable at 80 pixels
wide. Original design, not based on an existing comic, show, game, font, logo,
artist, or protected visual identity.
```
