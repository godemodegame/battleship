# Turn Token Prompt

## Runtime Asset

`prop-turn-token.glb`

## Prompt

Create an original 3D turn token for a mobile Battleship-style PvP game. The token should be a compact tactical marker that can hover near the active player's board or sit beside the UI. Use a dark metal base, cyan active-state light, amber waiting-state light, and red-magenta contested-state accent. The style is adult animated neo-noir graphic 3D with ink-like contours, painterly roughness, and high contrast.

The token should be recognizable as a current-turn indicator without relying on text. It can use abstract arrow-like geometry, a rotating ring, or a sharp signal wedge, but it must not include readable letters or symbols.

## Requirements

- Small readable prop.
- No text.
- Distinct active, waiting, and warning material states.
- Simple silhouette suitable for mobile.
- Works in 3D scene or UI overlay.
- Original design.

## Negative Prompt

No copied icon, no logo, no readable letters, no blockchain brand mark, no generic glowing orb, no cute board-game token, no beige material, no excessive detail.

## Notes

This asset should help players instantly understand whose turn it is.
