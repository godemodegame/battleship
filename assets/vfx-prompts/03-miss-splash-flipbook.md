- Runtime names: `vfx-miss-splash-01.png` through `vfx-miss-splash-08.png`
- Generate: 8 separate square RGBA images
- Source size: 1024x1024
- Runtime: 2048x1024 4x2 atlas

- Screen time: 0.65-0.9 seconds

## Base Prompt

```text
Create one isolated animation frame of a stylized cold-water miss effect for
an angled top-down mobile naval board game. Build the splash from graphic
pale-cyan wedges, cold-white droplets, dark petrol-blue ribbon shapes, and
thin electric-cyan contour accents. The water behaves like hand-painted
animation rather than fluid simulation: deliberately stepped motion, sharp
silhouette changes, broken brush edges, a narrow central crown, and long
asymmetric arcs that return toward a circular ripple. It must clearly read as
water, never as fire or magical energy, and must stay within roughly one board
cell.

Original adult animated neo-noir visual language, graphic 3D translated into
a hand-painted 2D VFX asset, bold readable silhouette, inked contour breakup,
dry-brush texture, selective halftone grain, hard shadow shapes, stepped
animation energy. Transparent background, fixed camera, fixed center point,
no environment, no text, no logo, no watermark.
```
