# Sunk-State Overlay

## Output

- Runtime names: `vfx-sunk-break-flash.png`,
  `vfx-sunk-residual-ink.png`
- Generate: 2 separate square RGBA images
- Source size: 1024x1024
- Runtime size: 512x512 each
- Blend: additive for break flash, alpha for residual ink

## Break Flash Prompt

```text
Create one isolated graphic destruction-break flash for the final hit that
sinks a ship in a stylized mobile naval game. A low horizontal silhouette
snaps apart into three dark angular masses. Between them, narrow hot-magenta
and impact-red fissures flare, crossed by a cold-white center cut and a thin
electric-cyan counter-edge. Add six to eight sharp debris slashes moving
outward, but keep the footprint compact and the center readable. The effect
is abstract and non-gory, like painted metal and signal light breaking in one
stepped animation frame. Transparent background, no complete ship model, no
scene, no text.
```

## Residual Ink Prompt

```text
Create an isolated top-down residual ink-and-smoke stain for a resolved sunk
cell. Use a compact asymmetric near-black shape with deep-petrol brush edges,
three transparent holes, sparse halftone erosion, and short glowing
hot-magenta cracks. A faint broken electric-cyan square edge appears only in
two corners, suggesting the tactical grid returning after the destruction.
The decal must feel persistent but quiet and must not hide the board cell.
Transparent background, no ship, no water scene, no text.
```

## Negative Prompt

```text
No realistic shipwreck, no casualties, no gore, no ocean disaster scene, no
large fireball, no full ship silhouette, no national markings, no text, no
logo, no magic symbol, no copied television frame.
```
