# Ship Battleship Prompt

## Runtime Asset

`ship-battleship.glb`

## Prompt

Create an original stylized 3D battleship for a mobile Battleship-style game. The ship should span three board cells and have a heavy, aggressive silhouette. Use a dark blue-black metal hull, electric cyan rim strips, red-magenta threat accents, and painterly roughness. The style should be adult animated neo-noir graphic 3D with bold contour lines, high contrast shadows, and cinematic neon reflections.

The battleship should have simplified turret-like forms and a strong central superstructure, but it must not copy a real-world warship. Make it feel like a fictional tactical miniature from a surreal encrypted ocean battlefield.

## Requirements

- Three-cell length silhouette.
- Heavy and powerful profile.
- Readable from top-down angled camera.
- Original fictional design.
- Mobile-friendly detail level.
- No readable text or symbols.

## Tripo Settings

Use these Tripo generation settings:

- AI Model: `v3.1 - Best Quality`
- Texture: `On`
- Texture Quality: `2K`
- PBR: `Off`
- Topology: `Triangle`
- Polycount: `3200`
- Ultra Mesh Quality: `Off`
- Generate in Parts: `Off`
- 8K Texture: `Off`
- Privacy: `Sharing Only`
- Runtime Target: lightweight mobile WebGL/browser asset with fast loading and low GPU cost

## Negative Prompt

No real ship replica, no existing franchise asset, no copied episode or show design, no flags, no logos, no readable numbers, no realistic military simulator finish, no cute low-poly shape, no excessive micro-detail.

## Notes

The battleship should contrast with the carrier by being narrower, heavier, and more weapon-focused.
