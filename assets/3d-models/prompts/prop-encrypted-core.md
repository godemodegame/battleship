# Encrypted Core Prompt

## Runtime Asset

`prop-encrypted-core.glb`

## Prompt

Create an original 3D prop that represents encrypted on-chain state for a mobile Battleship-style game. The object should look like a faceted sealed data core or compact ciphertext shard, not a generic glowing orb. Use dark glass, cyan internal light, amber pending-state accents, and subtle magenta warning edges. The style is adult animated neo-noir graphic 3D with high contrast, painterly roughness, and ink-like contours.

The prop can appear in lobby, fleet confirmation, or Fhenix-related UI moments. It should feel mysterious, secure, and tactical.

## Requirements

- Compact sealed data object.
- Faceted or capsule-like silhouette.
- No readable text, numbers, or symbols.
- Material slots for pending, confirmed, and error states.
- Mobile-friendly geometry.
- Works as a floating UI-adjacent prop or scene object.
