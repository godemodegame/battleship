# THOOM Launch Sprite Prompt

## Runtime Asset

`comic-sfx-thoom.png`

## Prompt

```text
Create one isolated 2D comic onomatopoeia sprite containing exactly the text
"THOOM!" for a heavy projectile launch in a mobile Battleship game. Use an
original adult animated neo-noir graphic illustration style with a
high-contrast painted-comic finish, heavy ink-like contours, painterly
roughness, dry-brush breakup, selective hatch marks, and stepped animation
timing.

Build a low, wide, weighty silhouette that surges from left to right. Compress
the T and H like a pressure kick, make the two O letters broad and slightly
offset like a recoil pulse, and taper the M toward the flight direction. Use a
signal-amber #FFB000 center, impact-red #FF3B30 pressure cuts, small cold-white
#E8F7FF highlights, and a thick near-black #07080D ink contour. Attach a short
dark recoil cloud and two angular amber launch streaks behind the lettering.
Keep the shape graphic and compact, not a realistic explosion.

The sprite must remain legible at mobile size and work beside the existing 3D
projectile. Transparent RGBA background, generous clear padding, 1536x768
source canvas, exact spelling "THOOM!", no other text. Original design, not
based on an existing comic, show, game, font, logo, artist, or protected visual
identity.
```

## Negative Prompt

```text
No extra text, no misspelling, no speech bubble, no rectangular panel, no
realistic cannon smoke, no fireball, no ship, no scene, no cute lettering, no
soft airbrush, no orange-only palette, no clean corporate vector style, no
watermark, no cropped edges.
```
