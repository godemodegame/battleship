# Projectile Trail Mask

## Output

- Runtime name: `vfx-projectile-trail-mask.png`
- Generate: one horizontal RGBA image
- Source size: 2048x512
- Runtime size: 512x128
- Blend: additive
- Mapping: repeat or stretch along a ribbon/tube behind the projectile

## Prompt

```text
Create an isolated horizontal projectile-trail texture for a fast stylized
energy shell in a mobile naval strategy game. The image runs left to right.
The right edge contains a tight cold-white spear tip; behind it is a narrow
signal-amber core streak, split by short hot-magenta cuts and surrounded by
thin electric-cyan brush filaments. The left two thirds dissolve into broken
dry-brush fibers, sparse halftone specks, and irregular transparent gaps.
The trail must feel fast, nervous, graphic, and hand-painted, with stepped
segments rather than one smooth gradient. Keep the central path straight and
the silhouette narrow so direction remains clear on a phone.

Original adult animated neo-noir visual language, hard graphic shapes,
painted breakup, inked edges, restrained cyan-magenta chromatic separation.
Transparent background, no scene, no object other than the trail, no text,
no logo, no watermark.
```
